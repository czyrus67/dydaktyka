Zadanie 41 – Teoria dotycząca pętli

Pętla for

Konstrukcja: 
for (inicjalizacja; warunek; krok) {
    // instrukcje
}
Opis:

Pętla for jest używana, gdy znamy liczbę powtórzeń. Składa się z trzech części: inicjalizacji, warunku i kroku.

Pętla while (z kontrolowanym wejściem)

Konstrukcja: 
while (warunek) {
    // instrukcje
}
Opis:

Pętla while sprawdza warunek przed każdym wykonaniem instrukcji. Jeśli warunek jest prawdziwy, wykonuje blok kodu.

Schemat blokowy:

Pętla do...while (z kontrolowanym wyjściem)

Konstrukcja: 
do {
    // instrukcje
} while (warunek);
Opis:

Pętla do...while wykona blok kodu przynajmniej raz, ponieważ warunek jest sprawdzany po wykonaniu instrukcji.
