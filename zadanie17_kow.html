1. Cztery sposoby wyprowadzania danych w JS:

  • document.write() - Wypisuje treść bezpośrednio do dokumentu HTML.
  • alert() - Wyświetla okno dialogowe z komunikatem.
  • console.log() - Wyświetla dane w konsoli przeglądarki.
  • innerHTML - Wstawia treść do elementu HTML.

2. Czym jest funkcja w JS?

  • Definicja: Zbiór instrukcji, które można wywoływać po nazwie.
  • Wykonanie: Po wywołaniu funkcja wykonuje swój kod.
  • Argumenty: Może przyjmować wartości wejściowe np. sum(a, b).
  • Return: Może zwracać wynik działania.

3. Metody pobierania elementów (tabela)

Metoda Opis
getElementById("id") Pobiera element o podanym ID.
getElementsByTagName("tag") Pobiera wszystkie elementy danego znacznika (np. "p").
getElementsByName("name") Pobiera elementy o danej nazwie.

4. Właściwości elementów HTML

  • innerHTML - Wstawia i odczytuje zawartość HTML danego elementu.
  • innerText - Odczytuje i zmienia czysty tekst w elemencie.
  • className - Ustawia lub zwraca nazwę klasy CSS elementu.
