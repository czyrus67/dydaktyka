Rozkład ułamka egipskiego na dwa takie same ułamki egipskie - KOD

Oto kod rozkładu ułamka egipskiego na dwa takie same ułamki:

Przykład dla m = 3:

        
            function rozkladUlamkaEgipskiego(m) {
                if (m == 3) {
                    return "1/3 = 1/6 + 1/6";
                }
            }
            
            console.log(rozkladUlamkaEgipskiego(3)); // "1/3 = 1/6 + 1/6"

Przykład dla m = 7:

        
            function rozkladUlamkaEgipskiego(m) {
                if (m == 7) {
                    return "1/7 = 1/14 + 1/14";
                }
            }
            
            console.log(rozkladUlamkaEgipskiego(7)); // "1/7 = 1/14 + 1/14"

Ten kod pokazuje, jak można rozłożyć ułamek egipski na dwa takie same ułamki. W przypadku 1/3 to 1/6 + 1/6, a dla 1/7 to 1/14 + 1/14.
