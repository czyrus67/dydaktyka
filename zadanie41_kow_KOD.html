    <h1>Zadanie 41 – Teoria dotycząca pętli</h1>

    <div class="petla">
        <h2>Pętla for</h2>
        <b>Konstrukcja:</b>
        <pre>
for (inicjalizacja; warunek; krok) {
    // instrukcje
}
        </pre>
        <b>Opis:</b>
        <p>Pętla for jest używana, gdy znamy liczbę powtórzeń. Składa się z trzech części: inicjalizacji, warunku i kroku.</p>
        <b>Schemat blokowy:</b><br>
        <img src="https://upload.wikimedia.org/wikipedia/commons/8/8c/For-loop-diagram.png" alt="Schemat blokowy pętli for">
    </div>

    <div class="petla">
        <h2>Pętla while (z kontrolowanym wejściem)</h2>
        <b>Konstrukcja:</b>
        <pre>
while (warunek) {
    // instrukcje
}
        </pre>
        <b>Opis:</b>
        <p>Pętla while sprawdza warunek przed każdym wykonaniem instrukcji. Jeśli warunek jest prawdziwy, wykonuje blok kodu.</p>
        <b>Schemat blokowy:</b><br>
        <img src="https://upload.wikimedia.org/wikipedia/commons/8/88/While-loop-diagram.png" alt="Schemat blokowy pętli while">
    </div>

    <div class="petla">
        <h2>Pętla do...while (z kontrolowanym wyjściem)</h2>
        <b>Konstrukcja:</b>
        <pre>
do {
    // instrukcje
} while (warunek);
        </pre>
        <b>Opis:</b>
        <p>Pętla do...while wykona blok kodu przynajmniej raz, ponieważ warunek jest sprawdzany po wykonaniu instrukcji.</p>
        <b>Schemat blokowy:</b><br>
        <img src="https://upload.wikimedia.org/wikipedia/commons/4/43/Do-while-loop-diagram.png" alt="Schemat blokowy pętli do...while">
    </div>
