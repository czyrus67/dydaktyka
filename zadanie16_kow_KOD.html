Sterowanie wydrukami - KOD

Oto kod JavaScript, który wykonuje obliczenia według podanego wzoru:

        
            var A = prompt("Podaj pierwszą liczbę A=");
            var B = prompt("Podaj drugą liczbę B=");

            var wynik = Math.pow(A, 3) + 3 * Math.pow(A, 2) * B + 3 * A * Math.pow(B, 2) + Math.pow(B, 3);

            document.write("<p class='granatowy'>Wczytane liczby: A=" + A + " B=" + B + "</p>");
            document.write("<p class='full-width'>(" + A + " + " + B + ")<sup>3</sup> = " + A + "<sup>3</sup> + 3(" + A + "<sup>2</sup>)(" + B + ") + 3(" + A + ")(" + B + "<sup>2</sup>) + " + B + "<sup>3</sup> = " + wynik + "</p>");

Ten kod prosi użytkownika o podanie dwóch liczb, a następnie wykonuje obliczenia na podstawie wzoru matematycznego i wyświetla wynik na stronie.
