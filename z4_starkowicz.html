Teoria JavaScript

1) Kiedy powstał JavaScript?

JavaScript powstał w 1995 roku.

2) Dlaczego jest to język skryptowy?

Jest to język skryptowy, ponieważ działa na poziomie interpretacji bez potrzeby kompilacji.

3) Gdzie jest wykonywany JavaScript (klient lub serwer)?

JavaScript jest wykonywany po stronie klienta, ale istnieje możliwość jego działania na serwerze (np. Node.js).

4) Dlaczego JS to bezpieczne narzędzie?

JS jest bezpieczny, ponieważ działa w izolowanej piaskownicy przeglądarki, uniemożliwiając dostęp do systemu operacyjnego.

5) Czy jest to język obiektowy? Jeśli tak, to dlaczego?

Tak, JavaScript jest językiem obiektowym, ponieważ wspiera obiekty, prototypy oraz ich dziedziczenie.

6) Wymień zalety JS:

  • Zaleta 1: Uniwersalność (działa w każdej przeglądarce).
  • Zaleta 2: Prosty w nauce i implementacji.
  • Zaleta 3: Duża społeczność i bogate biblioteki/frameworki.

7) Framework to:

Framework to zbiór gotowych narzędzi i bibliotek ułatwiających tworzenie aplikacji.

8) Do aplikacji mobilnych można użyć Framework:

React Native, Ionic, Flutter.

9) Aplikacje responsywne to:

Aplikacje, które dostosowują swój wygląd do różnych urządzeń i rozdzielczości ekranu.

10) <input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')">

Przycisk otwiera nowe okno przeglądarki z określoną stroną za pomocą funkcji WinOpen().

11) Instrukcja: window.open("obraz.html", "okienko", "toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200")

  • toolbar=no: Ukrywa pasek narzędzi przeglądarki.
  • directories=no: Ukrywa katalogi przeglądarki.
  • menubar=no: Ukrywa menu przeglądarki.
  • height=280: Ustawia wysokość okna na 280 pikseli.
  • width=160: Ustawia szerokość okna na 160 pikseli.
  • top=200: Pozycjonuje okno 200 pikseli od góry ekranu.
  • left=200: Pozycjonuje okno 200 pikseli od lewej krawędzi ekranu.

12) Instrukcja: window.close()

Zamyka obecne okno przeglądarki lub kartę.
